## 🏗️ NOUVELLE ARCHITECTURE - CAPACITÉS INDIVIDUELLES

### **Principe architectural** :
- **1 capacité = 1 classe Python dédiée** héritant de `BaseAbility`
- **Logique métier explicite** : Plus de parsing, que du code direct
- **Isolation complète** : Chaque capacité indépendante des autres
- **Maintenabilité optimale** : 1 bug = 1 fichier = 1 fonction

### **Mapping des données** :
- **Source** : `ability_names.csv` (59 lignes, 4 colonnes)
- **Structure** : `hero_code` (P-1 à P-8) + `ability_number` (1-6) + `generated_name` + `clean_description`
- **Cible** : 48 classes individuelles (8 héros × 6 capacités)

### **Gestion des cibles** :
```python
def _get_appropriate_targets(self, hero, ability):
    """Détermine les cibles selon le type de capacité"""
    
    # Capacités de soin → Alliés blessés
    if "soin" in ability.description.lower():
        allies = self._get_all_allies()
        return [a for a in allies if a.is_alive()]
    
    # Capacités de dégâts → Ennemis vivants  
    elif "dégât" in ability.description.lower():
        return self._get_alive_enemies()
    
    # Capacités de buff → Caster ou équipe
    elif "bonus" in ability.description.lower():
        return [hero] + self._get_all_allies()
    
    # Par défaut → Tous les combattants
    return self._get_all_allies() + self._get_alive_enemies()
```

### **Contexte de combat** :
```python
def _build_context(self, hero, ability):
    """Construit le contexte pour l'exécution de capacité"""
    return {
        'combat_round': st.session_state.get('combat_round', 1),
        'hero_health_percent': hero.current_health / hero.get_total_health(),
        'allies_count': len(self._get_all_allies()),
        'enemies_count': len(self._get_alive_enemies()),
        'spell_cost': getattr(ability, 'spell_cost', 0),
        'ability_uses_remaining': getattr(ability, 'uses_remaining', None)
    }
```
